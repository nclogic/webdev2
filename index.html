HTML Structure

HTML, which stands for HyperText Markup Language, is the foundation of web pages. It uses various tags to structure and organize content. Understanding the basics of HTML structure is crucial for web design.

  • Heading Tags (Not actual size, just sized to display sizes relative to each other.)

    • <h1>This is a 1st level heading</h1>
    • <h2>This is a 2nd level heading</h2>
    • <h3>This is a 3rd level heading</h3>
    • <h4>This is a 4th level heading</h4>
    • <h5>This is a 5th level heading</h5>
    • <h6>This is a 6th level heading</h6>

  • Links and Anchors

  • Lists (Ordered and Unordered)

  • Images and Media

  • Tables and Forms

  • Navigation

  • Here you will find a full list of available tags (Mozilla.org)

Doctype Declaration

This is the very first line in an HTML document. This lets the browser know what type of document format it is working with.

<!DOCTYPE html>  It is required as the first line. It is always followed by:

<html lang="en">  "en" is for English and is changed depending on the language the document is in.

For instance, "fr" is for French and "sp" is for Spanish.

This is Always the second line of the document, the beginning of the HTML structure.
